Courses

what you should learn?

HTML & CSS

Learn the basics of web development and create your first website. Master the art of structuring content and styling web pages. HTML5 semantics CSS3 styling and layouts Responsive design principles

Responsive Web Design

Create websites that look great on any device. Learn advanced CSS techniques and popular frameworks. Flexbox and Grid layouts Media queries CSS frameworks (Bootstrap, Tailwind)
